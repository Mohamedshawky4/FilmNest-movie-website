
# FilmNest 

A dynamic movie web app that lets users explore the latest films, view detailed information, and create personalized favorites lists. It integrates TMDb API for movie data and Firebase for secure user authentication and data storage.



## Features

- 🎥 Browse the latest and trending movies

- 🔍 Search for movies/TV-series by title

- ⭐ Add movies/TV-series to custom favorites lists

- 🔐 User authentication with secure login/logout


## Tech Stack

**Client:** JavaScript, CSS, HTML5

**Server:** FireBase, TMDB api
